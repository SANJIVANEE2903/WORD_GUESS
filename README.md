# 🎮 Word Guessing Game

A professional Python Word Guessing / Hangman Game built using Python and Tkinter GUI.

---

# 🚀 Features

- 🎨 Modern GUI Interface
- 🔤 Letter Guessing
- 🧠 Full Word Guessing
- 💡 Hint System
- ❤️ Attempts System
- 🏆 High Score Saving
- 🎭 Hangman Graphics
- 📂 Multiple Categories
- 🌙 Dark Theme UI

---

# 🛠️ Technologies Used

- Python
- Tkinter
- File Handling
- Random Module

# ▶️ How to Run

## Run Console Version

```bash
python main.py
```

## Run GUI Version

```bash
python gui_game.py
```

---

# 🎯 Game Rules

- Guess one letter at a time
- You can also guess the full word
- One hint available per game
- Wrong guesses reduce attempts
- Hint usage reduces score
